# Антропные курсы

Добро пожаловать на образовательные курсы Anthropic. В настоящее время этот репозиторий содержит пять курсов.  Мы предлагаем проходить курсы в следующем порядке:

1. [Основы антропного API](./anthropic_api_fundamentals/README.md) — обучает основам работы с Claude SDK: получение ключа API, работа с параметрами модели, написание мультимодальных подсказок, потоковая передача ответов и т. д.
2. [Интерактивное руководство по разработке подсказок] (./prompt_engineering_interactive_tutorial/README.md) — подробное пошаговое руководство по ключевым методам подсказок. [[Версия AWS Workshop](https://catalog.us-east-1.prod.workshops.aws/workshops/0644c9e9-5b82-45f2-8835-3b5aa30b1848/en-US)]
3. [Подсказки из реального мира] (./real_world_prompting/README.md) – научитесь включать методы подсказок в сложные подсказки из реального мира. [[Версия Google Vertex](https://github.com/anthropics/courses/tree/vertex/real_world_prompting)] 
4. [Быстрые оценки](./prompt_evaluations/README.md) — узнайте, как писать производственные оценки для измерения качества ваших подсказок.
5. [Использование инструментов] (./tool_use/README.md) — содержит все, что вам нужно знать для успешного использования инструментов в ваших рабочих процессах с Клодом.

**Обратите внимание, что на этих курсах часто отдается предпочтение нашей самой дешевой модели Claude 3 Haiku, чтобы снизить затраты на API для студентов, следящих за материалами. Если хотите, можете использовать другие модели Claude.**
