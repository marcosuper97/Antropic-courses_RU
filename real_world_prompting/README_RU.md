# Реальные подсказки

Добро пожаловать в подробное руководство Anthropic по подсказкам из реальной жизни. Этот курс предназначен для опытных разработчиков, которые уже окунулись в мир быстрого проектирования, особенно для тех, кто прошел наше подробное **[Интерактивное руководство по быстрому проектированию] (../prompt_engineering_interactive_tutorial/README.md)**. Если вы еще не прошли это руководство, мы настоятельно рекомендуем вам сделать это, прежде чем продолжить, поскольку оно обеспечивает углубленное изучение различных техник подсказок с практическими упражнениями.

За пять уроков вы научитесь включать ключевые методы подсказок в сложные, реальные подсказки. Мы рекомендуем вам начать с самого начала, с урока [Резюме подсказки](./01_prompting_recap.ipynb), поскольку каждый урок основан на ключевых понятиях, изложенных в предыдущих.
