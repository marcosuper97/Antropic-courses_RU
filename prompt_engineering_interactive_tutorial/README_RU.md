# Добро пожаловать в интерактивное руководство Anthropic по быстрому проектированию

## Введение и цели курса

Этот курс предназначен для того, чтобы дать вам полное пошаговое понимание того, как создавать оптимальные подсказки в Claude.

**После прохождения этого курса вы сможете**:
- Освоить базовую структуру хорошей подсказки 
- Распознать распространенные виды отказов и изучить методы «80/20» для их устранения.
- Понять сильные и слабые стороны Клода
- Создавайте сильные подсказки с нуля для распространенных случаев использования.

## Структура и содержание курса

Этот курс построен таким образом, чтобы у вас было много возможностей самостоятельно попрактиковаться в написании подсказок и устранении неполадок. Курс разбит на **9 глав с сопровождающими упражнениями**, а также приложение с еще более продвинутыми методами. Он предназначен для того, чтобы вы **проходили курс в порядке глав**. 

**Внизу каждого урока есть область «Площадка для примеров»**, где вы можете поэкспериментировать с примерами урока и увидеть своими глазами, как изменение подсказок может изменить ответы Клода. Также существует [ключ ответа](https://docs.google.com/spreadsheets/d/1jIxjzUWG-6xBVIa2ay6yDpLyeuOh_hR_ZB75a47KX_E/edit?usp=sharing).

Примечание. В этом уроке используется наша самая маленькая, быстрая и дешевая модель Claude 3 Haiku. У Anthropic есть [две другие модели] (https://docs.anthropic.com/claude/docs/models-overview), Claude 3 Sonnet и Claude 3 Opus, которые более умны, чем Haiku, причем Opus является самым умным.

*Это руководство также существует в [Google Sheets с использованием расширения Anthropic Claude for Sheets] (https://docs.google.com/spreadsheets/d/19jzLgRruG9kjUQNKtCg1ZjdD6l6weA6qRXG5zLIAhC8/edit?usp=sharing). Мы рекомендуем использовать эту версию, поскольку она более удобна для пользователя.*

Когда вы будете готовы начать, перейдите к «01_Basic Prompt Structure», чтобы продолжить.

## Содержание

Каждая глава состоит из урока и комплекса упражнений.

### Новичок
- **Глава 1:** Базовая структура приглашения

- **Глава 2:** Быть ясным и прямым  

- **Глава 3:** Назначение ролей

### Средний 
- **Глава 4:** Отделение данных от инструкций

- **Глава 5:** Форматирование вывода и выступление от имени Клода

- **Глава 6:** Предвидение (мышление шаг за шагом)

- **Глава 7:** Использование примеров

### Продвинутый
- **Глава 8:** Как избежать галлюцинаций

- **Глава 9:** Подсказки по созданию сложных объектов (примеры промышленного использования)
  - Сложные подсказки с нуля - Чат-бот
  - Комплексные запросы на юридические услуги
  – **Упражнение:** Сложные запросы для финансовых услуг.
  – **Упражнение:** Сложные подсказки для кодирования.
  - Поздравления и следующие шаги

- **Приложение:** Помимо стандартных подсказок
  - Цепочка подсказок
  - Использование инструментов
  - Поиск и извлечение
